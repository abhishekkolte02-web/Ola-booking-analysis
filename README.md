# Ola-booking-analysis
Ola Booking Analysis using SQL and Power BI
## Overview
This project analyzes ride booking data using SQL and Power BI.

## Tools Used
- SQL (MySQL)
- Power BI
- CSV Dataset

##  Files Included
- Dataset (CSV)
- SQL Queries
- Power BI Dashboard
- Project Report

##  Key Insights
- Booking trends and status analysis
- Revenue analysis
- Vehicle performance
- Cancellation reasons
- Customer behavior

##  How to Use
1. Import CSV into SQL
2. Run SQL queries
3. Open Power BI dashboard

## Outcome
This project demonstrates end-to-end data analysis:
Data → SQL → Dashboard → Insights
